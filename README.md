# 🧮 Simple Calculator Web App

A lightweight calculator built using HTML, CSS, and JavaScript.  
This project focuses on debugging, clean logic implementation, and handling common frontend issues.

---

## 🚀 Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Input validation (prevents empty values)
- Division by zero handling
- Clean and minimal UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)
